Bloodborne

Released in 2015, between Dark Souls 2 and Dark Souls 3, Bloodborne is a new spin on From Software tried and true formula. Bloodborne introduced Yarhnam, a Victorian setting which made stark contrasts to the medieval settings of the Dark Souls trilogy.
Bloodborne draws many similarities to the trilogy in that it also uses a set of starting classes and core stats (albeit using new classes and stats). Bloodborne's combat system showed large improvements when compared to the combat system found in the Dark Souls trilogy, providing players more options and playstyles.
